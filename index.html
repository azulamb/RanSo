Ranso

Ranso(らんそー)は入力欄に記入された項目をランダムに並び替えて表示するものです。発表順をランダムに決める際などにご利用ください。

Ranso will sort the input items at random.

使い方は以下の通りです。

  • All
    • 全てを並び替えて表示します。
  • Next
    • ボタンをクリックするたびに1件ずつ表示されます。
  • Reset
    • 表示を消します。
  • Clear
    • 表示と入力欄の中身を消します。

How to use.

  • All
    • Random sort & print.
  • Next
    • Print next item.
  • Reset
    • Delete items.
  • Clear
    • Delete items & clear input.
